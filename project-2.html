Application bureau en C#

Développement d'une application bureau en C# via le framework WinForms dans le cadre d'un projet pour l'entreprise fictive AngerScoot

Présentation du projet

Cette application a été le sujet d'un projet réalisé pendant ma première année de BTS SIO. Elle consiste en unue partie WEB et une partie Application à développer en Duo, et j'ai réalisé la partie Application. Ce programme doit permettre de gérer le parc de véhicules eléctriques de l'entreprise fictive AngerScoot, ainsi que toutes les personnes qui rechargent les véhicules. Cette application a été réalisée avec le langage C# et le framework Windows Forms.

Dans le cadre du projet, l'entreprise devait également avoir une base de données où stocker les informations sur les employés et le parc de véhicules. J'ai donc dû concevoir un Modèle conceptuel de Données (MCD), puis mettre en place un serveur pour héberger la base de données, puis enfin développer celle-ci.

Egalement, le projet étant réalisé en Duo, l'utilisation de Git, particulièrement GitLab a été primordiale pendant la réalisation du projet.

Pour finir, j'avais eu comme mission de concevoir une charte graphique que l'applicaion devait respecter. J'ai décidé de la créer via l'outil Canva.
