Application bureau en C#

Développement d'une application bureau en C# via le framework WinForms dans le cadre d'un projet pour l'entreprise fictive AngerScoot

Présentation du projet

Ce projet de première année de BTS SIO consistait à développer une application en C# avec Windows Forms pour "AngerScoot", une entreprise fictive. L'application, développée en duo, visait à gérer le parc de véhicules électriques et le personnel de recharge. J'étais en charge de la partie application de bureau.

Objectifs

L'objectif était de créer une solution de gestion complète, incluant une base de données pour stocker les informations sur les employés et les véhicules. Le projet exigeait également la conception d'une charte graphique et l'utilisation de Git pour la collaboration.

Déroulement et Apprentissages

Le développement a été réalisé en C# avec Windows Forms. J'ai conçu le modèle de données (MCD), mis en place un serveur SQL et développé la base de données. L'utilisation de GitLab a été essentielle pour la coordination du projet en duo. Ce projet m'a permis de développer des compétences en C#, en gestion de base de données SQL, et en collaboration via Git.
