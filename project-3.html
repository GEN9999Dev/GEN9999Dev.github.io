Site WEB Wordpress

Réalisation d'un site WEB complet via le CMS Wordpress, et d'une documentation utilisateur, dans le cadre d'un projet de cours pour l'entreprise fictive Aid'1 Jeun

Présentation du projet

Ce site WEB a été le sujet d'un projet réalisé pendant ma première année de BTS SIO. Il consiste en la réalisaion d'un site WEB hébergé sur un serveur virtuel sous Linux pour l'association fictive Aid'1 Jeun via l'outil Wordpress, puis de l'écriture d'une documentation technique pour permettre de maintenir et faire évoluer le site par le client.

Le site a été réalisé via Wordpress, mais l'uilisation d'extensions a été primordiale dans ce projet. J'ai utilisé le thème Astra, et les extensions Elementor, Yoast SEO, WooCommerce et UpdraftPlus. Ceci a permis d'avoir un rendu de site professionnel et complet, avec une boutique, une page d'accueil, et une page avec plusieurs tutoriels en rapport avec l'entreprise fictive.

J'ai dû également écrire une documentation technique complète pour aider les bénévoles de l'association à faire évoluer le site selon leurs préférences.

Ce projet m'a permis d'approfondir mes compétences en écriture de documentation, et m'a introduit aux CMS avec l'utilisation de Wordpress pour réaliser le site WEB.
