Site WEB Wordpress

Réalisation d'un site WEB complet via le CMS Wordpress, et d'une documentation utilisateur, dans le cadre d'un projet de cours pour l'entreprise fictive Aid'1 Jeun

Présentation du projet

Ce projet de première année de BTS SIO consistait à créer un site web pour l'association fictive "Aid'1 Jeun" en utilisant WordPress. Le site devait être hébergé sur un serveur virtuel Linux et accompagné d'une documentation technique pour la maintenance future par le client.

Objectifs

L'objectif principal était de développer un site web professionnel et complet, incluant une boutique en ligne, une page d'accueil et des tutoriels. Le projet nécessitait également la rédaction d'une documentation détaillée pour permettre aux bénévoles de l'association de gérer et de faire évoluer le site de manière autonome.

Déroulement et Apprentissages

Le site a été développé avec WordPress, en utilisant le thème Astra et des extensions clés comme Elementor, WooCommerce, et Yoast SEO. Ce projet m'a permis de me familiariser avec les CMS, d'approfondir mes compétences en rédaction de documentation technique et de comprendre comment construire un site web fonctionnel et optimisé pour le référencement.
